Embedding Master

Youtube

There are couple of ways to decorate your embedded Youtube video

  • By using autoplay=1, we can use the autoplay function for embedded Youtube vido. Don't forget adding mute=1 since Chrome browser will block playing the video.
  • By setting up start=seconds, we can play the embedded youtube video from certain time. I set start=180, so the Youtube video below starts at minute 3.
  • c_load_policy=1 allows embedded Youtube video to have cc.
  • modestbranding=1 remove the Youtube icon from your embedded Youtube video.
